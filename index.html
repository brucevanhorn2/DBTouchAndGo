Database Touch-and-Go

  • Why do it?

    Pilots use an exercise called "Touch and Go" to practice landings. In this exercise, the pilot takes off, circles the airport, then lands, then immediately takes off again, and repeats the exercise. Every pilot knows how to land, but you have to practice the skills in order to never forget your skills in those crucial times when it really matters.

    Here, I present a practice exercise to help you master the essential skills needed to work with the MySQL database from the command prompt. This isn't a tutorial - I expect you to already know what all the commands do, and why you'd use them. It's an exercise you can perform over and over to practice and acheive mastery of the important processes involved to creating and working with a MySQL database.

  • What should I already know?

    In order to be successful, you should:

    • Understand how to use the command line on your computer
    • Understand basic SQL data definition language (DDL) commands like create database and create table
    • Understand basic SQL data manipulation commands (DML) commands like select, insert, update, and delete
  • What do I need?

    To work with MySQL, you need:

    • The mysql client installed on your computer
    • The mysql command configured in your path such that it can be run from any folder
    • Have root access to a working non-production mysql database server including knowing the address, user name, and password. Many developers install MySQL on their development computer and access it via localhost, which is fine for this exercise.
