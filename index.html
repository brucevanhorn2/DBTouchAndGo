Database Touch & Go

In this touch and go, we'll practice connecting to and working with a very simple MySQL database.

  • Why do it?

    Pilots use an exercise called "Touch and Go" to practice landings. The pilot takes off, circles the airport, then lands, then immediately takes off again, sometimes with the wheels only briefly touching the ground. The pilot then repeats the takeoff and landing cycle, sometimes for hours at a time. Every pilot knows how to land, but you have to practice the skills in order to never forget your skills in those crucial times when it really matters.

    Here, I present a practice exercise to help you master the essential skills needed to work with the MySQL database from the command prompt. This isn't a tutorial - I expect you to already know what all the commands do, and why you'd use them. It's an exercise you can perform over and over to practice and acheive mastery of the important processes involved with creating and working with a MySQL database.

  • What should I already know?

    In order to be successful, you should:

    • Understand how to use the command line on your computer
    • Understand basic SQL data definition language (DDL) commands like create database and create table
    • Understand basic SQL data manipulation commands (DML) commands like select, insert, update, and delete
  • What do I need?

    To work with MySQL, you need:

    • The mysql client installed on your computer
    • The mysql command configured in your path such that it can be run from any folder
    • Have root access to a working non-production mysql database server including knowing the address, user name, and password. Many developers install MySQL on their development computer and access it via localhost, which is fine for this exercise.

    1 Open your command prompt.

    Connect to the MySQL Database

    mysql -u root -h localhost -p

    Substitute the user name and host address if necessary.
    3

    Show all available databases.

    show databases;
    4

    Create a new database.

    create database TouchAndGo;
    5

    Verify creation by running show databases again.

    show databases;
    6

    Use the TouchAndGo database.

    use TouchAndGo;
    7

    Check to see what tables exist in the database.

    show tables;
    8

    Create a table with an auto incrementing primary key field called ID.

    create table planes(
  plane_id int not null AUTO_INCREMENT,
  tail_number varchar(255),
  make varchar(255),
  model varchar(255),
  model_year INT,
  primary key (plane_id)
);
    9

    Verify the table was created using show tables again.

    show tables;
    10

    Insert some data into the table.

    insert into planes (tail_number, make, model, model_year) values ('ABC12356', 'Beechcraft', 'Bonanza', 1975);
    11

    Verify your insert by select all records in the table.

    select * from planes;
    12

    Insert a few more records. Don't forget you can use the up arrow on your keyboard to bring back earlier commands, and then just edit them.

    insert into planes (tail_number, make, model, model_year) values ('DEF12356', 'Bombardier', 'Learjet 60 XR', 2012);
    into planes (tail_number, make, model, model_year) values ('GHI12356', 'Lockheed Martin', 'XR-71 Blackbird', 1969);
    insert into planes (tail_number, make, model, model_year) values ('DEF12356', 'Bombardier', 'Learjet 60 XR', 2012);
    insert into planes (tail_number, make, model, model_year) values ('GHI12356', 'Cessna', 'Lattitude', 2017);
    13

    Verify your inserts by select all records in the table.

    select * from planes;
    14

    Update the record with the ID of 3

    update planes set tail_number = 'XYZ3456' where plane_id = 3;
    15

    Verify your update by selecting just the record with the ID you updated.

    select * from planes where plane_id = 3;
    16

    Select the count of all the records in the table.

    select count(*) as total_planes from planes;
    17

    Delete the record with an ID equal to .

    delete from planes where plane_id = 2;
    18

    Verify the delete by selecting the count again. There should be one fewer.

    select count(*) as total_planes from planes;
    19

    We're done! Let's clean up. Drop the database.

    drop database TouchAndGo;
    20

    Exit the mysql prompt.

    exit;
    21

  • What should I learn next?
    Once you've mastered this exercise, you should:
    • Learn more advanced sql commands operating on a single table.
    • Learn to use inner join to join two or more tables
    • Learn summarizing queries using count, average, and sum functions
    • Learn to use group by and having clauses
    • Learn to substitute some default value in place of null if a field value is null
