1. Open your command prompt.

  2. Connect to the MySQL Database

    mysql -u root -h localhost -p

    Substitute the user name and host address if necessary.

  3. Show all available databases.

    show databases;

  4. Create a new database.

    create database TouchAndGo;

  5. Verify creation by running show databases again.

    show databases;

  6. Use the TouchAndGo database.

    use TouchAndGo;

  7. Check to see what tables exist in the database.

    show tables;

  8. Create a table with an auto incrementing primary key field called ID.

    create table planes;

  9. Verify the table was created using show tables again.

    show tables;

  10. Insert some data into the table.

    insert into planes...;

  11. Verify your insert by select all records in the table.

    select * from planes;

  12. Insert a few more records.

    select * from planes;
